\chapter{Sample Difference Implementation}
\label{app:diffsrc}

We present a \textbf{perl} language implementation of our basic text
matching algorithm.
Function \texttt{edit\_diff} computes the difference between two
lists of words, and returns
a list of edit-script operations consisting of
\textbf{Ins}, \textbf{Del}, and \textbf{Mov} operations.

There are two implementations presented here.
The \textbf{WikiTrust::BasicDiff} module presents the bare essence of how our
differencing algorithm works.
Module \textbf{WikiTrust::FasterDiff} simplifies the amount of work
computed by assuming that longer matches are preferred over shorter
matches, and only calculates the shorter matches when a longer match
doesn't fit due to earlier choices.
