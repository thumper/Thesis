\chapter{Sample Difference Implementation}
\label{app:diffsrc}

We present a \textbf{perl} language implementation of our basic text
matching algorithm.
Function \texttt{edit\_diff} takes two lists of words, and returns
a list of edit-script operations consisting of
\textbf{Ins}, \textbf{Del}, and \textbf{Mov} operations.
